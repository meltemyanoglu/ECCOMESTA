\section{Non Functional Requirements}
% Nonfunctional requirements: what other characteristics should the system have, besides functional requirements?



The system should; 
\begin{itemize}
    \item Respond in acceptable time for user queries or transactions. 
    \item Documented clearly and be upgraded to new versions.; 
    \item Work with other software systems and databases through standardized interfaces and protocols.
    \item Internationalization to allow customers speaking various languages to utilize the system.
    


\end{itemize}
