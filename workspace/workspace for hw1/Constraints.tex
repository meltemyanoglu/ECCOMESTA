\section{Constraints}
%What constraints should our system respect? 

The DBMS application should satisfy the following constraints:

\begin{itemize}
    \item The employees of a store can see the products' quantities of another store but cannot change them. 
    \item For transferring goods, both store managers have to approve the transfer.
    \item 

    \item Be implemented with PostgreSQL;
    \item Client side implemented using html, css, javascript and jquery;
    \item Server side implemented using Tomcat, java servlet, JSP, MVC and REST web services;
    \item Operating system: Linux.


\end{itemize}
